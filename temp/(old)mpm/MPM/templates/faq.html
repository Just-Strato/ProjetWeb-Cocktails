A quoi sert ce site?

Notre site vous permet de suivre très simplement votre progression dans le visionage d'une série ou d'obtenir des informations comme le synopis de cette série

Comment se servir du site?

Vous pouvez utilisze le site sans compte pour voir les informations sur les séries et vous pouvez vous inscrire en quelques secondes pour plus de fonctionnalitées depuis l'onglet Inscriptions

Comment ajouter une série à mon compte?

Vous pouver ajouter une série à votre compte en accédant à la vue détaillée en cliquant sur son titre depuis l'ognet série

Comment supprimer une série de mon compte?

Vous pouvez supprimer une série de votre compte en sélectionnant "---" dans l'avancement depuis la vue détaillée de cette série

Comment modifier mon avancement/note?

Vous pouvez modifier ces informations depuis la vue détaillée d'une série ou augmenter le dernier épisode vu de 1 depuis votre page personelle grace au "+" une série "A regarder" passeras automatiquement en "cours"

Comment ça marche?

Nous utilisons la très bonne api de Betaseries disponible ici pour les séries et les comptes sont stockés sur une base MySQL

Pourquoi utiliser notre site?

Grace à nous, vous n'oublierez plus jamais ou vous étiez arrivés la dernière fois que vous avez regardé votre série préférée et vous pourrez découvrir de nouvelles séries populaires
